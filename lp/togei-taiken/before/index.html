✨ この体験が素敵な3つの理由

Three Special Points

🌟 4名限定の特別な空間

最大4名の少人数制だから、講師がお一人おひとりに素敵な時間を提供します。初めての方でも安心して素晴らしい作品づくりに集中できる、特別な環境です。

🎡 電動ろくろで本格体験

手びねりだけでなく電動ろくろも体験できる素晴らしいワークショップ。プロと同じ設備で、素敵な器づくりの醍醐味を存分に味わえます。

📦 焼き上がり後ご自宅へ配送

制作した作品は講師が丁寧に焼成。約1ヶ月後、素敵に仕上がった素晴らしい作品をご自宅までお届けします。届いた瞬間の感動は特別なものです。

About the Host

藤村 美咲（ふじむら みさき）

🏺 陶芸家 / アトリエ「土と風」主宰

多摩美術大学工芸学科卒業後、信楽にて3年間修行。2014年より代々木上原にアトリエ「土と風」を構え、制作活動とワークショップを行う。素敵な作品は国内外のギャラリーで展示され、これまでに8回の素晴らしい個展を開催。「土に触れることで生まれる素敵な対話」をテーマに、初心者から経験者まで幅広く特別な陶芸の魅力を伝えている。陶芸歴12年。

❓ よくあるご質問

FAQ

陶芸は全くの初めてですが大丈夫ですか？ +
はい、もちろんです！参加者の約8割が初心者の方です。基本からしっかりレクチャーしますし、4名限定なのでお一人おひとりに丁寧にサポートいたします。素敵な作品が必ず完成しますのでご安心ください。✨
作品はいつ届きますか？ +
焼成に約3〜4週間お時間をいただきます。素晴らしく焼き上がった作品は、丁寧に梱包してご自宅まで配送いたします（配送料は参加費に含まれています）。届いた時の感動は特別なものですよ！📦
どんな服装で行けばいいですか？ +
汚れても気にならない服装でお越しください。エプロンはこちらでご用意しております。爪は短めに切っておくと作業しやすいです。素敵な制作タイムをお楽しみください！🎨
電動ろくろと手びねり、どちらがおすすめですか？ +
初めての方には手びねりが取り組みやすいですが、電動ろくろも素晴らしい体験です。当日、講師と相談しながらお好きな方法をお選びいただけます。両方試してみるのも素敵ですよ！🎡
キャンセルはできますか？ +
3日前までのキャンセルは全額返金いたします。2日前〜当日のキャンセルは参加費の50%をキャンセル料としていただきます。日程変更は空きがあれば対応可能ですので、お気軽にご相談ください。📞
